# Trae Rules Configuration

本仓库用于配置和管理各类 Trae 的规则，旨在为不同的 C# 开发场景提供标准化的开发规范和 AI 辅助规则。

## 目录结构

- `userrule/`: 存放按日期格式留档的用户规则。
- `project-rules/`: 存放针对不同项目类型的详细规则描述。

## 规则分类

目前主要涵盖以下 C# 架构：
- **WinForm**: 传统桌面应用开发规范。
- **WPF (MVVM)**: 现代桌面应用开发，强调数据绑定与架构分离。
- **WebAPI (MVC)**: 后端服务开发，遵循 RESTful 风格。
- **Blazor**: 基于 .NET 的 Web 交互式 UI 开发。
- **MAUI**: 跨平台移动与桌面应用开发。

## 使用说明

将对应的规则文件内容复制到 Trae 的配置中，即可实现针对特定项目类型的 AI 辅助优化。
